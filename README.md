# Orbis Advisory — CV en ligne

CV en ligne moderne et responsive de **Clément Boizard**, Chef de Projet IT & Product Owner Freelance.

## Aperçu

Site one-page livré en **un seul fichier `index.html`** (CSS et JS embarqués), hébergeable directement sur GitHub Pages sans configuration.

## Sections

- **Hero** — Effet terminal/typewriter animé + proposition de valeur
- **À propos** — Présentation, photo initiales, 3 compteurs animés
- **Services** — 4 cartes : Pilotage IT, Product Ownership, Transformation digitale, Conduite du changement
- **Parcours** — Timeline interactive (Galderma, CEGEMA, AGPM, Crédit Agricole)
- **Compétences** — Tags par catégorie (Méthodes Agile, Outils, Secteurs, Certifications)
- **Références** — Logos clients stylisés
- **Contact** — Formulaire, email, LinkedIn, localisation
- **Footer** — Copyright Orbis Advisory 2026

## Stack technique

- HTML5 / CSS3 / JavaScript vanilla
- Polices : [Syne](https://fonts.google.com/specimen/Syne) + [DM Sans](https://fonts.google.com/specimen/DM+Sans) via Google Fonts
- Aucun framework, aucune dépendance backend

## Mise en ligne via GitHub Pages

1. Merger la branche vers `main`
2. Aller dans **Settings → Pages**
3. Source : branche `main`, dossier `/root`
4. Le site est accessible sur `https://boizardclement-art.github.io/SiteWebVitrine/`

## Visualisation locale

Ouvrir `index.html` directement dans un navigateur, ou lancer un serveur local :

```bash
python3 -m http.server 8080
```

Puis ouvrir `http://localhost:8080`.
